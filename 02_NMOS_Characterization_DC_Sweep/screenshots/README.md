This folder contains the screenshots used in the documentation of **Part 02 – NMOS Characterization Using DC Sweep Analysis**.

The images illustrate the complete design flow, including:

- Project setup
- NMOS device selection
- Voltage source configuration
- Lab pin placement
- Completed schematic
- Simulation code block
- Generated SPICE netlist
- NgSpice simulation
- IDS–VDS characteristics
- IDS–VGS characteristics

These screenshots are referenced throughout the `README.md` file to explain each stage of the design and simulation process.
