# Sky130 CMOS Inverter Design

## Overview

This repository documents the complete design and analysis flow of a CMOS inverter using the Sky130 Open-Source Process Design Kit (PDK). The project covers schematic design, circuit simulation, performance analysis, and layout implementation using open-source EDA tools.

## Tools Used

- Sky130 Open-Source PDK
- Xschem
- NgSpice
- Magic VLSI

## Repository Structure

The repository is organized into different stages of the CMOS inverter design flow. Each section contains the required design files, simulation results, screenshots, and observations.

```
01_CMOS_Inverter_Design_and_Transient_Analysis/
02_Voltage_Transfer_Characteristic/
03_Noise_Margin_Analysis/
04_Propagation_Delay_Analysis/
05_Power_Analysis/
06_CMOS_Inverter_Layout/
07_DRC_Verification/
08_LVS_Verification/
```

> This repository is intended to document my implementation and understanding of the CMOS inverter design flow using the Sky130 open-source PDK.
