# 01 – CMOS Inverter Design and Transient Analysis

## Description

This section presents the design and transient simulation of a CMOS inverter using the Sky130 Open-Source PDK. The inverter schematic is created in Xschem and simulated using NgSpice to verify its switching behavior by analyzing the transient response of the input (VIN) and output (VOUT).

---

## Objective

- Design a CMOS inverter using Sky130 devices.
- Generate the SPICE netlist.
- Perform transient simulation using NgSpice.
- Verify inverter operation by plotting VIN and VOUT waveforms.
