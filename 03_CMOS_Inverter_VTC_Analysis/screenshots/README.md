# Screenshots – CMOS Inverter Voltage Transfer Characteristics (VTC)

This folder contains the screenshots used in the documentation of **Part 03 – CMOS Inverter Voltage Transfer Characteristics (VTC)**.

The images illustrate the complete workflow, including schematic creation, symbol generation, testbench setup, simulation configuration, netlist generation, and the final Voltage Transfer Characteristic (VTC) curve obtained using **Xschem** and **NgSpice**.

## Screenshot List

| Screenshot | Description |
|------------|-------------|
| `cmos_pt3_choosing_nfet.png` | Selecting the NMOS transistor from the Sky130 library. |
| `cmos_pt3_choosing_pfet.png` | Selecting the PMOS transistor from the Sky130 library. |
| `cmos_pt3_creating_input_pin.png` | Creating the input (VIN) pin. |
| `cmos_pt3_creating_output_pin.png` | Creating the output (VOUT) pin. |
| `cmos_pt3_schematic.png` | Completed CMOS inverter schematic. |
| `cmos_pt3_symbol_created_from_the_schematic.png` | Symbol generated from the CMOS inverter schematic. |
| `cmos_pt3_created_testbench.png` | Testbench created using the inverter symbol. |
| `cmos_pt3_netlist.png` | Generated SPICE netlist for the testbench. |
| `cmos_pt3_simulation_output.png` | Voltage Transfer Characteristic (VTC) simulation result showing VOUT versus VIN. |
